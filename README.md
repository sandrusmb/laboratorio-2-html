## Bootcamp JS 2 - Laboratorio Módulo 1 - HTML

### Primeros pasos

- Clonar el proyecto.
- Instalar las dependencias con `npm install`.
- Ejecutar `npm run dev`.
- Abrir el navegador en el puerto `http://localhost:5173/`
